# 1999-website

A "port" of my website made for _[Like it's 1999](https://99.hackclub.com/)_. It is made entirely (exept for 88x31 buttons) in Windows 98 using EditPlus for code and Intenet Explorer to veiw. To use it in Windows 98 import the file `1999-state_26-07-11_22-25.bin.gz` to [99.hackclub.com](https://99.hackclub.com/emulator/).
